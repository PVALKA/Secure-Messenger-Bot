# 🚀 ربات پیام ناشناس پیشرفته تلگرام (Cloudflare Workers)

یک ربات حرفه‌ای و مدرن پیام ناشناس تلگرام که بر بستر **Cloudflare Workers** و پایگاه داده **Cloudflare D1** پیاده‌سازی شده است. این ربات دارای سیستم هوشمند قفل کانال (جوین اجباری)، پنل مدیریت پیشرفته شیشه‌ای، و سیستم تعاملی پاسخ/بلاک کاملاً یکپارچه است.

## ✨ قابلیت‌ها و ویژگی‌ها
- ⚡️ **سرعت و پایداری بالا:** اجرا روی لبه شبکه (Edge) کلودفلر (بدون نیاز به سرور و هاست).
- 🔐 **امنیت و حریم خصوصی:** مسیرهای امن برای Webhook بر اساس هش توکن ربات.
- 📢 **سیستم جوین اجباری (Force Join) هوشمند:** 
  - بررسی دقیق وضعیت ادمین بودن ربات در کانال.
  - حذف خودکار کانال از دیتابیس در صورت لغو دسترسی ادمین.
- ⚙️ **پنل مدیریت درون برنامه‌ای:** مدیریت قفل کانال‌ها به صورت مستقیم در ربات با دستور `/admin`.
- 🛡 **مدیریت کاربران:** قابلیت بلاک/آن‌بلاک فرستنده‌های ناشناس و اعلام وضعیت خوانده شدن پیام (Seen).
- 🗄 **دیتابیس ابری رایگان:** استفاده از دیتابیس رابطه‌ای D1 کلودفلر.

---

## 🛠 پیش‌نیازها
1. یک حساب کاربری رایگان در [Cloudflare](https://dash.cloudflare.com/).
2. نصب بودن `Node.js` و `npm` روی سیستم.
3. یک ربات تلگرام (دریافت توکن از [@BotFather](https://t.me/BotFather)).
