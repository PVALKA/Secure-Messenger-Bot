# 🤖 Telegram Anonymous Bot (Cloudflare Workers & D1)

**یک ربات پیام ناشناس پیشرفته و کاملاً ابری (بدون نیاز به سرور و هاست!)**

این پروژه یک ربات تلگرامی هوشمند، امن و بسیار سبک است که صفر تا صد آن روی شبکه لبه **Cloudflare Workers** و دیتابیس **Cloudflare D1** اجرا می‌شود. برای راه‌اندازی این ربات نیازی به سیستم، سرور یا دانش برنامه‌نویسی ندارید و تمام مراحل مستقیماً از طریق مرورگر و پنل سایت کلودفلر انجام می‌شود.

---

## ✨ امکانات
- ⚡️ **بدون نیاز به سرور (Serverless):** اجرای کاملاً رایگان، پرسرعت و پایدار روی لبه شبکه کلودفلر.
- 🗄 **دیتابیس ابری (Cloudflare D1):** استفاده از دیتابیس رابطه‌ای SQL قدرتمند برای ذخیره دائمی اطلاعات بدون حذف شدن.
- 📢 **قفل کانال هوشمند (Force Join):** سیستم جوین اجباری با بررسی لحظه‌ای؛ در صورت لغو ادمینی ربات، کانال خودکار از سیستم حذف می‌شود.
- ⚙️ **پنل مدیریت شیشه‌ای:** افزودن، حذف و مشاهده کانال‌های قفل اجباری مستقیماً از داخل ربات با دستور `/admin`.
- 🛡 **آنتی‌اسپم و بلاک:** قابلیت ریپلای مستقیم روی پیام‌های ناشناس و بلاک/آن‌بلاک کردن مزاحمان بدون افشای هویت.
- 🔒 **مسیر وب‌هوک امن:** استفاده از هش توکن برای ساخت آدرس وب‌هوک مخفی جهت جلوگیری از درخواست‌های فیک.

---

## 🚀 آموزش راه‌اندازی مستقیم در داشبورد کلودفلر

### 📌 پیش‌نیازها
1. توکن ربات تلگرام (دریافت از [@BotFather](https://t.me/BotFather))
2. آیدی ربات شما (بدون `@`)
3. آیدی عددی تلگرام ادمین (دریافت از ربات‌هایی مثل `@userinfobot`)
4. یک اکانت رایگان در [Cloudflare](https://dash.cloudflare.com/)

---

### 🛠 مراحل نصب و استقرار

#### مرحله ۱: ساخت دیتابیس (D1)
1. وارد داشبورد کلودفلر شوید. از منوی سمت چپ به بخش **Storage & Databases** و سپس **D1 SQL Database** بروید.
2. روی **Create database** کلیک کنید و یک نام (مثلاً `anonymous-db`) برای آن بگذارید.
3. روی دیتابیسی که ساختید کلیک کنید و به تب **Console** بروید.
4. کد SQL زیر را در کادر متنی پیست کرده و دکمه **Execute** را بزنید تا جداول دیتابیس ساخته شوند:

```sql
CREATE TABLE IF NOT EXISTS users (
  telegram_id TEXT PRIMARY KEY,
  rkey TEXT,
  username TEXT,
  name TEXT,
  admin_state TEXT DEFAULT '',
  blocked TEXT DEFAULT '',
  target_user TEXT DEFAULT '',
  reply_to_user TEXT DEFAULT '',
  reply_to_msg_id TEXT DEFAULT ''
);

CREATE TABLE IF NOT EXISTS channels (
  chat_id TEXT PRIMARY KEY
);
```

#### مرحله ۲: ساخت سرویس (Worker)
1. از منوی سمت چپ به بخش **Workers & Pages** بروید.
2. روی دکمه **Create application** و سپس **Create Worker** کلیک کنید.
3. یک نام برای ورکر خود انتخاب کرده و روی **Deploy** کلیک کنید.

#### مرحله ۳: اتصال دیتابیس (Binding)
1. وارد صفحه Worker خود شوید و به تب **Settings** بروید.
2. از منوی کناری وارد بخش **Bindings** شوید.
3. روی **Add** کلیک کرده و **D1 database** را انتخاب کنید.
4. در کادر `Variable name` **دقیقاً** کلمه `DB` را با حروف بزرگ بنویسید.
5. در کادر `Database`، دیتابیسی که در مرحله اول ساختید را انتخاب کرده و روی **Deploy** کلیک کنید.

#### مرحله ۴: افزودن متغیرهای محیطی (Variables)
1. در همان تب **Settings**، به بخش **Variables and Secrets** بروید.
2. روی **Add** کلیک کنید و متغیرهای زیر را دقیقاً با همین نام‌ها وارد کنید:
   - `BOT_TOKEN`: توکن ربات شما
   - `BOT_USERNAME`: آیدی ربات شما (مثلاً `MyNashenasBot`)
   - `OWNER_ID`: آیدی عددی تلگرام شما
3. تنظیمات را **Deploy** کنید.

#### مرحله ۵: جایگذاری کد ربات
1. در صفحه ورکر، روی دکمه **Edit code** (بالا سمت راست) کلیک کنید.
2. تمامی کدهای پیش‌فرض را پاک کنید.
3. کدهای فایل اصلی این ریپازیتوری را کپی کرده و در این قسمت پیست کنید.
4. دکمه **Deploy** (یا Save and deploy) را بزنید.

#### مرحله ۶: فعال‌سازی ربات (تنظیم Webhook)
در کدهای این ربات سیستم تنظیم خودکار و امن وب‌هوک تعبیه شده است. برای اتصال ربات به تلگرام، کافیست آدرس ورکری که کلودفلر به شما داده است را همراه با `/init` در مرورگر خود باز کنید.

فرمت لینک به این شکل است:
```text
https://YOUR_WORKER_NAME.your-username.workers.dev/init
```
*(آدرس ورکر خود را می‌توانید در صفحه اصلی ورکر کپی کنید. فقط کافیست `/init` را به انتهای آن بچسبانید و در مرورگر باز کنید).*

✅ اگر در مرورگر پیام زیر را مشاهده کردید، ربات با موفقیت فعال شده است:
`Webhook Status: {"ok":true,"result":true,"description":"Webhook was set"}`

## 👑 راهنمای استفاده ادمین
پس از استقرار، با اکانت تلگرام خود (همان `OWNER_ID`) وارد ربات شوید و دستور زیر را ارسال کنید:
```text
/admin
```
پنل شیشه‌ای مدیریت باز خواهد شد و می‌توانید کانال‌های جوین اجباری را اضافه یا حذف کنید. ربات به طور خودکار صحت ادمین بودن خود را در کانال‌ها بررسی می‌کند.
